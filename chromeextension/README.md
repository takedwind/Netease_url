# 网易云音乐智能助手 Chrome 插件

这是一个 Manifest V3 独立插件，不依赖 Flask、Docker 或 NAS 后端服务。

## 功能

- 自动识别当前网易云音乐歌曲、歌单、专辑页面
- 弹窗内解析单曲信息、封面、歌词和播放/下载地址
- 搜索歌曲
- 解析歌单曲目
- 解析专辑曲目
- 右键菜单解析/下载网易云链接
- 在网易云页面右下角注入快捷按钮
- 使用浏览器下载 API 将歌曲、封面、歌词打包为 zip
- 内置健康检查和接口信息页

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录：`chromeextension`

## 使用说明

- 打开网易云歌曲页、歌单页或专辑页后，点击插件图标即可自动识别。
- 也可以在弹窗里手动输入歌曲 ID、歌单 ID、专辑 ID 或对应链接。
- 下载走网易云网页外链和 Chrome 下载能力，会生成包含音频、封面、歌词和信息文件的 zip，不会写入 NAS 目录。

zip 内容示例：

```text
歌手 - 歌名/
├── 歌手 - 歌名.mp3
├── cover.jpg
├── lyrics.lrc
├── translated-lyrics.lrc
└── info.json
```

## 说明

独立插件版本使用网易云网页接口和当前浏览器会话。标准播放/下载通常可用，高音质解析能力会受到账号状态、版权状态和网易云接口限制。
